Creo.JS Tests

Test LinkDescription
script_engine_testing Simple web tool for interactive testing of Creo.JS programs with simple JavaScript editor and output console. The page supports loading and saving test scripts from user script library initially populated with sample scripts. It is very handy during Creo.JS learning and development process
creojs_framework_tests Various Creo.JS framework tests. Some tests require active assembly in session
init_term_creojs Illustrates Creo.JS page loading and unloading events
active_model_name Simple page that displays active model file name. Requires active model in session
ui_mouse_click_methods Compares mouse related APIs between pure Web.Link and Creo.JS
compare_weblink_and_v8 Compares performance of assembly traversal between pure Web.Link and Creo.JS. Requires active assembly in session
explore_creo_icons Lists image files from Creo install resource directory. Provides image files filter to find icons by name fragments
load_demo_app Loads demo application which sets model listeners and menu commands. See the page for further instructions
promises_test Tests promises mechanism for calling a chain of Creo.JS functions. Promises allow chaining multiple asynchronous Creo.JS calls one after another. The test collects the number of coordinate system features for all models in the session and displays it along with the number of the session models. Requires active part in session. If the current model is an assembly Creo.JS code throws an exception stopping the chain execution
